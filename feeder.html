Smart Pet Feeder

Why is this project important?

The Smart Pet Feeder is designed to automatically dispense food for pets at scheduled times or when they approach the feeder. This ensures that pets are well-fed, even when their owners are not at home. It also prevents overfeeding by controlling portion sizes. The system is particularly beneficial for busy pet owners and can be used for dogs, cats, and even birds. Additionally, it helps in maintaining hygiene as food is only dispensed when needed, reducing contamination and waste.

Components Needed

  • Arduino Uno
  • Ultrasonic Sensor (HC-SR04)
  • Servo Motor (SG90 or MG995)
  • Jumper Wires
  • 5V Power Supply
  • Plastic Container (for food storage)
  • Small Pipe or Funnel (to direct food)
  • Breadboard (for testing connections)

How to Build the Smart Pet Feeder

Step 1: Connect the Ultrasonic Sensor

The ultrasonic sensor is used to detect when the pet is near the feeder. Connect the **VCC** to 5V, **GND** to GND, **Trigger Pin** to Arduino **Pin 2**, and **Echo Pin** to **Pin 3**.

Step 2: Connect the Servo Motor

The servo motor controls the food dispenser. Connect the **VCC** to 5V, **GND** to GND, and the **Signal Pin** to **Pin 9** on the Arduino.

Step 3: Program the Arduino

Upload the following code to your Arduino: 

#include <Servo.h>
Servo foodServo;
const int trigPin = 2;
const int echoPin = 3;

void setup() {
    foodServo.attach(9);
    pinMode(trigPin, OUTPUT);
    pinMode(echoPin, INPUT);
    Serial.begin(9600);
}

void loop() {
    digitalWrite(trigPin, LOW);
    delayMicroseconds(2);
    digitalWrite(trigPin, HIGH);
    delayMicroseconds(10);
    digitalWrite(trigPin, LOW);

    int duration = pulseIn(echoPin, HIGH);
    int distance = duration * 0.034 / 2;

    if (distance < 20) {
        foodServo.write(124); // Open
        delay(2000);
        foodServo.write(9);   // Close
    }

    delay(500);
}

Step 4: Assemble the Feeder

- Cut a hole in a plastic container for food dispensing.
- Attach the servo motor to a small flap that opens and closes.
- Place the ultrasonic sensor at the front to detect the pet.
- Mount all components securely in a stable position.

Step 5: Power & Test

Power the system using a 5V supply and test it by moving an object (or a pet) near the sensor. The feeder should open when the pet is detected and close after 2 seconds.
