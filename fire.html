Fire Alarm System

Why is this project important?

The Fire Alarm System is designed to detect smoke or fire and alert users in case of an emergency. This project uses sensors to monitor the environment and activate an alarm when smoke levels exceed a safe threshold. It is crucial for ensuring safety in homes, offices, and public spaces by providing early warnings before a fire becomes dangerous.

Components Needed

  • Arduino Uno
  • MQ-2 Gas Sensor
  • Buzzer
  • Jumper Wires
  • 5V Power Supply
  • LED (Optional for visual indication)

How to Build the Fire Alarm System

Step 1: Connect the MQ-2 Gas Sensor

The MQ-2 gas sensor detects smoke or gas. Connect the **VCC** to 5V, **GND** to GND, **Analog Pin** to **A0** on the Arduino.

Step 2: Connect the Buzzer

The buzzer will alert users when smoke is detected. Connect the **VCC** to 5V, **GND** to GND, and the **Signal Pin** to **Pin 8** on the Arduino.

Step 3: Program the Arduino

Upload the following code to your Arduino: 

#include <MQ2.h>

int smokePin = A0;  // Pin for MQ-2 sensor
int buzzerPin = 8;  // Pin for the buzzer
int smokeLevel = 0; // Variable to store the smoke level reading

void setup() {
    pinMode(buzzerPin, OUTPUT);
    Serial.begin(9600);
}

void loop() {
    smokeLevel = analogRead(smokePin);
    Serial.println(smokeLevel);
    
    // Check if the smoke level is high enough
    if (smokeLevel > 400) {
        digitalWrite(buzzerPin, HIGH); // Turn on the buzzer
    } else {
        digitalWrite(buzzerPin, LOW);  // Turn off the buzzer
    }

    delay(1000);
}

Step 4: Assemble the System

- Mount the MQ-2 sensor in a well-ventilated area.
- Connect the buzzer where it can be heard easily.
- Make sure the sensor is placed high to detect smoke properly.

Step 5: Power & Test

Power the system using a 5V supply. Test by producing smoke near the sensor or using a lighter to check if the alarm goes off.
