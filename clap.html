Clap to Turn On Light

Why is this project important?

The **Clap to Turn On Light** system is a fun and convenient way to control the lighting in a room. It works by detecting a clap sound through a microphone sensor, making it easier to turn on lights without physically pressing any buttons or switches. This project can be used in smart home systems and helps demonstrate how simple sound-based control systems work.

Components Needed

  • Arduino Uno
  • Sound Sensor (Microphone Module)
  • LED
  • 220-ohm Resistor
  • Jumper Wires
  • Breadboard
  • 5V Power Supply

How to Build the Clap to Turn On Light System

Step 1: Connect the Sound Sensor

The sound sensor detects sound in the environment. Connect the **VCC** to 5V, **GND** to GND, and the **OUT Pin** to Arduino **Pin 2**.

Step 2: Connect the LED

The LED will light up when a clap is detected. Connect the **positive (long)** leg of the LED to **Pin 13** on the Arduino and the **negative (short)** leg to GND through a 220-ohm resistor.

Step 3: Program the Arduino

Upload the following code to your Arduino: 

const int soundSensorPin = 2;  // Sound sensor pin
const int ledPin = 13;         // LED pin

void setup() {
  pinMode(ledPin, OUTPUT);  // Set LED pin as an output
  pinMode(soundSensorPin, INPUT);  // Set sound sensor pin as input
  Serial.begin(9600);  // Initialize serial communication for debugging
}

void loop() {
  int soundValue = digitalRead(soundSensorPin);  // Read sound sensor
  if (soundValue == HIGH) {  // If sound is detected
    digitalWrite(ledPin, HIGH);  // Turn on LED
    delay(2000);  // Keep LED on for 2 seconds
    digitalWrite(ledPin, LOW);  // Turn off LED
  }
}

Step 4: Assemble the Circuit

- Connect the components on a breadboard.
- Place the sound sensor on the breadboard and wire it to the Arduino.
- Connect the LED with the appropriate resistor.
- Ensure all connections are secure before powering the Arduino.

Step 5: Test the System

After uploading the code, clap your hands near the sound sensor. The LED should turn on for 2 seconds and then turn off. If the system doesn't work as expected, check the wiring and make sure the sound sensor is placed in an appropriate location for detecting claps.
