Water Level Alert System

Why is this project important?

The **Water Level Alert System** is essential for monitoring water levels in tanks or reservoirs. It uses a water level sensor to detect whether the tank is full or low. The system alerts the user using a buzzer when the water reaches a certain level, helping to prevent overflows or dry tank conditions. This is particularly useful in water management systems.

Components Needed

  • Arduino Uno
  • Water Level Sensor
  • Buzzer
  • Jumper Wires
  • Breadboard
  • 5V Power Supply

How to Build the Water Level Alert System

Step 1: Connect the Water Level Sensor

The water level sensor detects the water level in the tank. Connect the **VCC** to 5V, **GND** to GND, and the **OUT Pin** to Arduino **Pin 2**.

Step 2: Connect the Buzzer

The buzzer will sound when the water level reaches the desired point. Connect the **positive leg** of the buzzer to **Pin 13** on the Arduino and the **negative leg** to GND.

Step 3: Program the Arduino

Upload the following code to your Arduino: 

const int waterLevelPin = 2;  // Water level sensor pin
const int buzzerPin = 13;     // Buzzer pin

void setup() {
  pinMode(buzzerPin, OUTPUT);  // Set buzzer pin as output
  pinMode(waterLevelPin, INPUT);  // Set water level sensor pin as input
  Serial.begin(9600);  // Initialize serial communication for debugging
}

void loop() {
  int waterLevel = digitalRead(waterLevelPin);  // Read water level sensor
  if (waterLevel == HIGH) {  // If water level is detected
    digitalWrite(buzzerPin, HIGH);  // Activate the buzzer
    delay(1000);  // Buzzer sound duration
    digitalWrite(buzzerPin, LOW);  // Deactivate the buzzer
  }
}

Step 4: Assemble the Circuit

- Place the water level sensor in the water tank and wire it to the Arduino.
- Connect the buzzer with the appropriate wiring.
- Make sure all connections are secure before powering up the system.

Step 5: Test the System

After uploading the code, place the water level sensor at different heights in the water tank. The buzzer should sound when the water reaches the sensor. If it doesn't work, check your wiring and ensure the sensor is submerged correctly.
