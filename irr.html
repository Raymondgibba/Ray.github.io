Smart Irrigation System

Why is this project important?

The **Smart Irrigation System** automatically waters plants based on the soil moisture level, ensuring that your plants are always properly hydrated without over-watering. This system is perfect for gardeners who want to save time, conserve water, and optimize the health of their plants.

Components Needed

  • Arduino Uno
  • Soil Moisture Sensor
  • Relay Module
  • Water Pump
  • Jumper Wires
  • Power Supply
  • Water Reservoir

How to Build the Smart Irrigation System

Step 1: Connect the Soil Moisture Sensor

The soil moisture sensor detects the moisture level in the soil. Connect the **VCC** to 5V, **GND** to GND, and the **A0 Pin** (analog output) to **A0** on the Arduino.

Step 2: Connect the Relay Module

The relay will control the water pump. Connect the **VCC** to 5V, **GND** to GND, and the **IN Pin** to **Pin 8** on the Arduino.

Step 3: Connect the Water Pump

The water pump will be powered using the relay. Connect the **positive terminal** of the pump to the **NO (Normally Open) Pin** of the relay and the **negative terminal** to the **GND**.

Step 4: Program the Arduino

Upload the following code to the Arduino: 

const int soilMoisturePin = A0;   // Soil moisture sensor pin
const int relayPin = 8;            // Relay control pin

void setup() {
  pinMode(relayPin, OUTPUT);  // Set relay pin as output
  Serial.begin(9600);  // Initialize serial communication for debugging
}

void loop() {
  int soilMoistureValue = analogRead(soilMoisturePin);  // Read soil moisture level
  Serial.print("Soil Moisture: ");
  Serial.println(soilMoistureValue);

  if (soilMoistureValue < 400) {  // If the soil is dry
    digitalWrite(relayPin, HIGH);  // Activate the relay (turn on the water pump)
    Serial.println("Watering...");  // Output status
  } else {
    digitalWrite(relayPin, LOW);  // Deactivate the relay (turn off the water pump)
    Serial.println("Soil Moisture is sufficient.");  // Output status
  }
  delay(2000);  // Wait for 2 seconds before reading again
}

Step 5: Assemble the Circuit

- Place the soil moisture sensor into the soil in your garden or plant pot.
- Connect the water pump to the relay and power it using the relay's output.
- Ensure your water reservoir is connected to the pump to supply water to the plants.

Step 6: Test the System

After uploading the code, place the soil moisture sensor into the soil. If the soil is dry, the water pump will activate and water the plants. Once the moisture level is sufficient, the pump will turn off automatically.
